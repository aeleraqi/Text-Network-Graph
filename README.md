Visualize relationships between words, entities, or concepts in text using **network graphs**.

## 📖 About

**Text Network Graph** transforms raw text into interactive network visualizations. By treating words as nodes and co-occurrences as edges, this tool reveals hidden patterns in text corpora.

## ✨ Features

- Build word co-occurrence networks from any corpus
- Named Entity Relationship (NER) graphs
- Centrality analysis (degree, betweenness, PageRank)
- Interactive Plotly/NetworkX visualizations
- Community detection in text networks
- Export as HTML or PNG

## 🚀 Getting Started

```bash
git clone https://github.com/aeleraqi/Text-Network-Graph.git
pip install networkx plotly spacy pandas
jupyter notebook
```
